# Práctica JS Game 2D

Proyecto de desarrollo de videojuego 2D orientado a la práctica y consolidación de habilidades en JavaScript utilizando HTML5 Canvas.

## 🛠️ Tecnologías utilizadas

- **JavaScript (ES6)**
- **HTML5 Canvas**
- **HTML + CSS**

## 🎯 Objetivos de aprendizaje

- Consolidar conocimientos de JavaScript aplicado a videojuegos
- Implementar un sistema de renderizado usando Canvas
- Desarrollar un sistema de mundo por cuadrantes (8x8)
- Aplicar generación procedural basada en coordenadas
- Implementar mecánicas como movimiento, disparo, colisiones y habilidades
- Practicar organización y estructura de proyectos

## 🌍 Sistema de mapa

El juego utiliza un sistema de mundo dividido en cuadrantes:

- Mapa 8x8
- Cada cuadrante es una zona independiente
- Generación procedural por coordenadas
- Navegación entre cuadrantes

## ▶️ Ejecución

Abrir el archivo `index.html` en el navegador
